📤 データエクスポート
このサイトの評価・編成・解説データを全訪問者に見せるには、以下の手順が必要です。
  1. 各カードの詳細ページで「✏️ 編集」してデータを入力・保存する
  2. このページの「エクスポート」ボタンを押して editorial-data.js をダウンロード
  3. ダウンロードしたファイルを cards/ フォルダに配置してGitHubにコミット
  4. GitHub Pagesにデプロイされると、全訪問者にデータが反映される ✅
